## Sistema Integral de Gestión de Inventario y E-commerce

Este proyecto documenta mi proceso de aprendizaje para construir un ERP completo utilizando Python. El objetivo es crear un sistema escalable que gestione inventario físico mediante lectores de códigos de barras, ventas en local (POS) y una tienda virtual con verificación de pagos.

### Fases del Proyecto

#### Fase 1: Lógica de Consola
- Configuración del entorno y Git.
- Prototipo de lectura de códigos de barras.
- Algoritmo de detección de productos (Nuevo vs Existente).

#### Fase 2: Base de Datos
- Implementación de SQLite.
- Registro de historial de movimientos.

#### Fase 3: Aplicación Web (Usaremos Django)
- Estructura del servidor.
- Panel de administración.

#### Fase 4: Tienda virtual y Pagos
- Tienda pública.
- Verificador de pagos por transferencia.

### Tecnologías a utilizar
- Python 3
- Git & GitHub
- Django, SQLite, HTML/JS (Próximamente)
